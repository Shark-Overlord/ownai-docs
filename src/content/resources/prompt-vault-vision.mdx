## 项目定位

Prompt Vault Vision 是 OwnAI 的第一个资源项目，定位为本地视觉 Prompt 资产管理器。

它不是简单的收藏夹，而是围绕“Prompt + 效果图 + 来源页面 + 匹配证据 + 标签 + 本地检索”建立的一套长期资产系统。适合持续收集图像生成、视频生成、Web UI、AI Skill、Agent 工具和 MCP 服务相关的开源仓库与案例。

## 解决什么问题

- 把分散在 GitHub、页面截图和仓库说明里的视觉 Prompt 统一归档。
- 给 Prompt 绑定效果图、来源页面、匹配证据和筛选状态，避免只保存一段文本后失去上下文。
- 对 Web UI 组件库、设计规范仓库、Skill 仓库进行分类收藏和画像评分。
- 用本地 SQLite 做索引，图片和导出文件保存在本地目录，方便离线维护和迁移。
- 支持定时搜索、仓库扫描、标注任务和运行记录，减少人工重复整理。

## 核心能力

- GitHub 增量发现：按图像生成、视频生成、Web UI、Skill 等分类搜索相关仓库。
- 仓库去重：对 URL、Fork、相似内容和图片 hash 做归一化处理。
- Prompt 资产库：保存 Prompt、效果图、来源页面、证据链、翻译、标签和筛选状态。
- Web UI 资产库：收藏前端组件库、设计规范和 UI 仓库，并按画像得分排序。
- Skill 资产库：整理 AI Skill、Agent 工具、MCP 服务和具体使用场景。
- 本地 UI：通过 React 管理瀑布流、详情抽屉、筛选、收藏、搜索和导出。
- 后台任务：支持定时搜索、仓库扫描任务、标注队列和任务运行记录。

## 技术栈

- 后端：Python、FastAPI、SQLite、Pydantic、LangGraph
- 前端：React、TypeScript、Vite、Tailwind CSS、React Query、Zustand
- 数据：SQLite、本地图片资产、Markdown / JSON / CSV 导出

## 适合场景

- 建立自己的视觉 Prompt 案例库。
- 收集高质量图像生成、视频生成和 UI 生成 Prompt。
- 维护 Web UI、组件库、设计系统和 Skill 仓库参考。
- 给 AI 设计、AI 编码、Agent 工具沉淀可检索的素材池。
- 在本地保存数据，不依赖第三方 SaaS 管理资产。
